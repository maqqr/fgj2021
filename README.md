Development
-----------

First time setup:

Install recent version of node and npm first.

```
npm install
npm rebuild node-sass
```

Start local development server (`http://localhost:5000`):

```
npm run dev
```

Making a final build (appears in `dist` directory):

```
npm run dev build
```
